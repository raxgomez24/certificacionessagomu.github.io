# 🎓 Sagomu - Centro de Certificaciones Internacionales

Catálogo completo de certificaciones internacionales con temarios detallados y sistema de agendamiento.

## 🌐 Características

- **75 certificaciones** internacionales disponibles
- **4 temarios completos** con información detallada
- **Sistema de agendamiento** funcional
- **Diseño responsive** para todos los dispositivos
- **Navegación fluida** entre páginas

## 🚀 Tecnologías

- HTML5 Semántico
- CSS3 con Variables y Grid
- JavaScript ES6+
- JSON para datos
- Diseño Mobile-First

## 📱 Páginas Principales

- **Catálogo**: certificaciones.html
- **Temarios**: temario.html?cert=[id]
- **Agendar**: agendar.html?cert=[id]
- **Demo**: DEMO-SISTEMA.html
